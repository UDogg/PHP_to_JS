## <a id="StageMaster"></a>Stage Master
- master for creating the stage to map with sub stages 
- mapping and stage creation both done in same page [stagemaster](https://uatdashboard-revamp.fynity.in/stagemaster)
- we can also able to set priority for the stages to be shown according to priority in policy status report page 
- All the sub stages are comes from mongodb for the first time we need to hit the [sync-url](https://uatdashboard-revamp.fynity.in/sync-data)
