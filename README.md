# ⚡ PHP Native Compiler (PHC)

> Compile a subset of PHP into native executables — no interpreter, no *AMP stack, just raw performance.

---

## 🧠 Overview

**PHC (PHP Native Compiler)** is a custom-built compiler that translates PHP code into C, compiles it into a native binary, and executes it locally.

Unlike traditional PHP execution (which relies on an interpreter and web server stack), PHC provides:

- 🚀 Native execution speed  
- 🧱 Full control over runtime behavior  
- 🧩 Modular compiler architecture  
- 🖥️ Desktop IDE interface via Electron  

This project is both:
- a **learning exercise in compiler construction**
- and a **practical systems-level toolchain**

```plaintext

│
├── .gitignore                # Excludes binaries, node_modules, temp files
```

## ⚙️ Features

### ✅ Compiler Pipeline
- PHP → Tokenization → Parsing → Code Generation → C → Native Binary

### ✅ Custom Runtime System
- Dynamic typing via tagged unions (`Value`)
- Runtime functions (`php_echo`, `php_add`, etc.)
- Extensible architecture for PHP-like behavior

### ✅ GUI IDE
- Code editor
- Compile & run button
- Output display
- Local execution (no deployment)

### ✅ Native Execution
- No Apache / Nginx
- No PHP interpreter
- Direct system-level execution
